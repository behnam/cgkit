:mod:`glslangtokenize` --- Tokenizer for the OpenGL 2 shading language
======================================================================

.. module:: cgkit.glslangtokenize
   :synopsis: Tokenizer for the OpenGL 2 shading language


This module provides a lexical scanner for the OpenGL 2 shading language. You
use it in the same way as the Python tokenize module is used.
