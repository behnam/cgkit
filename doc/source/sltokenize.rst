:mod:`sltokenize` --- Tokenizer for the RenderMan Shading Language
==================================================================

.. module:: cgkit.sltokenize
   :synopsis: Tokenizer for the RenderMan Shading Language

This module provides a lexical scanner for the RenderMan Shading Language. You
use it in the same way as the Python :mod:`tokenize` module is used.
